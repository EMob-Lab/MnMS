# Analyze scripts

---

## Path output file

This Python script analyze a path output CSV file generated by MnMS.

The script reads a CSV file with path data, performs integrity checks, and reports statistics about the quality and 
completeness of the paths.

### Features 

- Reads path output data from a CSV file (semicolon-separated).
- Prints first and last departure times.
- Analyzes and counts:
  - Users with infinite path cost.
  - Users with no path defined.
  - Users with infinite path length.
  - Users without mobility services defined.
  - Users without a chosen path.
- Outputs a summary report in the terminal.

### Usage example

````bash 
python analyze_path_output.py path.csv
````

## User output file

This Python script analyze an user output CSV file generated by MnMS.  

The script reads a CSV file with user data, performs consistency checks, and reports statistics about user states 
and missing information.

### Features

- Reads user activity data from a CSV file (semicolon-separated).
- Prints earliest and latest activity times.
- Analyzes and counts:
  - Users without a link.
  - Users without a position.
  - Users in a deadend state.
  - Users who have arrived at their destination.
- Outputs a summary report in the terminal.

### Usage example

````bash 
python analyze_user_output.py user.csv
````
